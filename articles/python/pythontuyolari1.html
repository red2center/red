Python: Az Bilinen Ama İşe Yarar İpuçları ​

Python, güçlü ve esnek bir programlama dili olarak hem yeni başlayanlar hem de uzmanlar için birçok fırsat sunar. Ancak, Python'un bazı özellikleri ve püf noktaları genellikle gözden kaçar. Bu makalede, Python'da az bilinen ama oldukça işe yarar birkaç ipucunu ve kod örneklerini paylaşacağız.

1. zip ile Tersine Çevirme ​

zip fonksiyonu, listeleri veya diğer iterable'ları birleştirirken çok kullanışlıdır. Ancak, aynı zamanda tersine çevirme işlemi için de kullanılabilir!

Örnek: ​

python
keys = ['a', 'b', 'c']
values = [1, 2, 3]

# İki listeyi bir sözlüğe çevirin
dictionary = dict(zip(keys, values))
print(dictionary)  # {'a': 1, 'b': 2, 'c': 3}

# Ters çevirme işlemi
reversed_keys_values = dict(zip(dictionary.values(), dictionary.keys()))
print(reversed_keys_values)  # {1: 'a', 2: 'b', 3: 'c'}

2. collections.defaultdict ile Varsayılan Değerler ​

Sözlüklerde bir anahtar bulunmadığında hata almak yerine, varsayılan bir değer atamak çok işe yarar.

Örnek: ​

python
from collections import defaultdict

# Varsayılan değer list olan bir sözlük
default_dict = defaultdict(list)

default_dict['key1'].append(1)
default_dict['key1'].append(2)
default_dict['key2'].append(3)

print(default_dict)  # defaultdict(<class 'list'>, {'key1': [1, 2], 'key2': [3]})

3. Farklı Dosya Formatlarını Kolayca Okuma ve Yazma: pandas ​

pandas kütüphanesi sadece veritabanı işlemleri için değil, aynı zamanda CSV, Excel ve JSON gibi dosya formatlarını hızlıca okuma ve yazma için de idealdir.

Örnek: ​

python
import pandas as pd

# Bir CSV dosyasını okuyun
df = pd.read_csv('data.csv')

# Verileri inceleyin
print(df.head())

# Yeni bir Excel dosyası olarak kaydedin
df.to_excel('output.xlsx', index=False)

4. Lambda ile Koşullu Liste Filtreleme ​

Listelerde belirli koşullara göre filtreleme yapmak için lambda fonksiyonlarını kullanabilirsiniz.

Örnek: ​

python
numbers = [1, 2, 3, 4, 5, 6]

# Sadece çift sayıları filtreleyin
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4, 6]

5. with Bloğu ile Kaynak Yönetimi ​

Dosya işlemleri yaparken kaynak yönetimi için with bloğu çok işe yarar. Bu blok, otomatik olarak kaynakları temizler.

Örnek: ​

python
with open('example.txt', 'w') as file:
    file.write('Merhaba, Dünya!')

# Dosya otomatik olarak kapanır, hata riskini azaltır.

6. enumerate ile Daha İyi Döngüler ​

enumerate fonksiyonu, döngülerde indeksleri yönetmek için çok kullanışlıdır.

Örnek: ​

python
languages = ['Python', 'JavaScript', 'C++']

for index, language in enumerate(languages, start=1):
    print(f"{index}: {language}")
# 1: Python
# 2: JavaScript
# 3: C++

7. Python Debugging: pdb ​

Hata ayıklamak için Python'un yerleşik pdb modülünü kullanabilirsiniz.

Örnek: ​

python
def buggy_function(x):
    import pdb; pdb.set_trace()  # Hata ayıklama noktasını ayarla
    return x + 1

buggy_function(5)

8. Listeleri Daha Hızlı İşlemek için itertools ​

itertools kütüphanesi, iterasyon işlemlerini optimize etmek için harikadır.

Örnek: ​

python
from itertools import combinations

# Bir listenin tüm kombinasyonlarını bulun
items = ['a', 'b', 'c']
for combo in combinations(items, 2):
    print(combo)
# ('a', 'b')
# ('a', 'c')
# ('b', 'c')

Sonuç ​

Python'un esnekliği, dilin pek çok az bilinen ama güçlü özelliklerini içerir. Bu püf noktaları ve kod blokları, hem zaman kazanmanızı sağlar hem de kodlarınızı daha verimli hale getirir. Bu özellikleri projelerinizde kullanarak, Python ile daha etkili bir şekilde çalışabilirsiniz.
